# The naikan-test module

This module provides a set of tools and utilities for testing the naikan modules.


Maven Usage
-------------------

```xml
<dependency>
    <groupId>com.enofex</groupId>
    <artifactId>naikan-test</artifactId>
    <version>latest version</version>
</dependency>
```
